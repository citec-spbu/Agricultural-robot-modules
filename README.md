Начальная версия сервиса-симулятора робота, предназначенная для тестирования систем навигации, сбора телеметрии и обработки данных от сенсоров.
Сервис учитывает спецификацию датчиков робота и реализует передачу данных в бинарном формате, что приближает поведение к реальному устройству.

Симулятор создан на базе FastAPI и предоставляет API для управления движением робота и получения его текущего состояния.
